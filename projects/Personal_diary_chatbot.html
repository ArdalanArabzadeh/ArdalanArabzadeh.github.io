Personal Diary Chatbot

A local AI diary assistant that remembers your daily notes and answers questions about your past — fully private and offline.

GitHub View source on GitHub

Project Overview

A local AI diary assistant that remembers your daily notes and answers questions about your past — fully private and offline. Entries can be added directly from Telegram and recalled through a natural, first-person chat interface powered by a dual-retriever RAG system.

Project Details

  • Type: Local AI App + Telegram Bot
  • Tech Stack: Python, Streamlit, LangChain, Ollama, ChromaDB, BM25, Telegram Bot API

Description

The Personal Diary Chatbot is a local, privacy-first AI assistant designed to act as your digital memory companion. It stores daily notes written as .txt files and allows you to naturally ask questions such as “What did you do on 03.02.2025?” or “Tell me about a day you felt scared.”

In each session run, Every diary entry is automatically chunked, date-tagged, and indexed into a Chroma vector database for semantic retrieval, enhanced with a BM25 keyword retriever for precise results. The chatbot then responds in a personal, first-person tone — as if it’s recalling its own memories.

You can also add your daily entries directly from Telegram (as long as the bot script is running locally on your computer), making journaling effortless even when you’re away from your computer. The bot automatically creates a dated .txt file in your local diary folder, ready to be indexed by the chatbot.

Motivation

I built this project as a way to create a personal “memory companion” — an assistant that keeps track of my thoughts and experiences in my own words. It’s a small but meaningful step toward preserving my memories and creating something that could one day speak from my perspective, based entirely on my own experiences and writing.

Features

  • Fully local and private — no cloud or API calls
  • Dual-retriever RAG (semantic + keyword) for accurate recall
  • Date-based filtering and natural date recognition
  • First-person, memory-style responses
  • Telegram integration for easy daily journaling

How to Use

  1. Clone the project repository from GitHub
  2. Install Ollama from https://ollama.com/.
  3. Write your daily memory as a .txt file named DD.MM.YYYY.txt inside the diary/ folder.
  4. Or, send your memory via the connected Telegram bot.
    1. Follow the Telegram bot instructions written on README.md file of Github repository and create a .env file with your personal configurations.
    2. Start the bot locally: 
      python telegram_bot.py
    3. With the bot running, send your memory in Telegram. The bot will ask whether it belongs to today’s date or another date, then automatically create/save the .txt file in your diary/ folder named by that date.
  5. Run the app locally: 
    streamlit run app.py
    Then open http://localhost:8501 in your browser to access the interface.

Chat naturally — for example:

“What did you do on 20.10.2025?”

“Tell me about a fun day.”

Docker

Alternatively, you can easily deploy the Telegram bot project using Docker. There is a pre-built Docker image available in my public Docker Hub repository. Simply clone or pull the Docker image and follow the setup steps described in the README.md file of the project's GitHub repository.
